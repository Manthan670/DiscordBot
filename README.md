# Problem it solves :
Reducing friction in accessing Discord server and user information by converting multi-step UI navigation into instant text commands that is basic  automation + observability.

---

## How to run the scripts:
1. Clone the repository to your local machine using the command:
   ``` git clone https://github.com/AdityaAmanAir/DiscordBot ```
2. Navigate to the cloned directory:
   ``` cd /DiscordBot ```
3. Add the Token from the discord development in the variable token in main/java/Bot.java
4. Compile the JAVA by running:
   ``` javac -cp "JDA-5.2.2-withDependencies.jar" src/main/java/Bot.java ```
5. Run the byte code file using the command:
   ``` java -cp ".:JDA-5.2.2-withDependencies.jar:src/main/java" Bot ```

And All Set, use the bot where - whenever by inviting it into the GUILD(Server).

## Required Tools:
- Need to install Java Development Kit (JDK)
- JDA (Java Discord API)
